# @optomitron/opg

Optimal Policy Generator — ranks and scores policies using causal evidence, welfare functions, and Bradford Hill criteria.

**Paper:** [Optimal Policy Generator](https://opg.warondisease.org) — Evidence-based policy ranking using causal confidence scores and multi-method weighting.

**Related:**
- [Optimocracy](https://optimocracy.warondisease.org) — Two-metric welfare function
- [dFDA Specification](https://dfda-spec.warondisease.org) — Causal inference engine (dependency)
